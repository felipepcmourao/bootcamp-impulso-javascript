# Dimensão e Alinhamento

Para trabalhar com isso há algumas propriedades.

#### width
Largura.

#### height
Altura.

#### max-width
Largura máxima.

#### max-height
Altura máxima.

#### margin
Espaçamento entre elementos. Serve para alinhar algum elemento automaticamente.

#### text align
Para alinhar algum elemento automaticamente.
